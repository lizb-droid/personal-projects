Workout List Screen
Entry point. Full week view — 7 day rows showing session name, completion state, and today's position. Today's row expands inline to show exercises + Continue/Start CTA. Tap any other row to expand it. Phase progress bar + scrollable week tabs sit above the list.
Phase Progress Bar
Pattern: Phase label + "Wk N / N" right. Segmented bar below — one pill per week. Completed = indigo. Current = pink (slightly taller + glow halo). Future = muted #333. Gap between segments = 4px. Sits on --dk2.
Phase 2 · Build Wk 6 / 8
Day card states (upcoming, today, completed, disabled) are documented in Cards → Week View. Week tab component is documented in Navigation → Week Tabs.
Bottom Actions
Hierarchy: Add exercise = primary dashed CTA (44px). Export + Import = secondary ghost (40px), separated by border-top. Utility — never compete with the exercise list.

Exercise Detail Screen
Active set logging. Accessed by tapping any row in the List view. Shows exercise name, all sets, and auto-launches the rest timer after each set is checked.
Header + Target Weight
Rule (all screens): Date small on top, exercise name large below. PR tag inline with name. Target weight row: Label "Target" + large indigo number + calculation context (% × PR value). Shown as reference — user taps set row to log actual weight.

Rest Timer Screen
Trigger: Auto-launches after any set checkbox is checked. Full dedicated screen — not a modal overlay. Light frosted-glass card on dark bg. Recommended duration pre-selected by exercise type.
Full Timer Layout
Composition: Light glass card overlaid on exercise context (set pills visible behind). Dismiss × → exercise name → countdown → ± 10s → 4 preset pills → Start + Reset. See Cards → Rest Timer for the full interactive specimen.
Active preset: pink border + pink tint bg. No check icon on preset (removed — avoids confusion with set checkboxes). Tapping resets countdown.
