## Autoregulation Rules

- **Add load when RPE ≤ 8**
- **Deload trigger:** Squat Set 1 at RPE 8+ when target is 6–7 → treat the full session as a deload
- **Power work FIRST:** Clean Pull / Box Jump before squats; KB Swing / Push Press before pressing. Power quality degrades before strength quality.
- **Luteal phase:** Same load, drop a set if needed, no new PRs. Friday Upper B is sacrificial under high fatigue — protect Monday and Tuesday.

---

## Postpartum Monitoring

- Film Squat Set 4 every other week — check depth, knee tracking, core integrity
- Watch Copenhagen Plank progression — indicator of pelvic floor + hip stability
- Prolactin elevated → estrogen suppressed → joint laxity possible under heavy loads
- **Pelvic floor symptoms = stop and regress immediately. No exceptions.**

---

## Red Lines — Flag Immediately

- Persistent soreness >7/10 for 3+ days
- Joint instability or unexpected joint pain
- Pelvic floor symptoms of any kind
- Weight drop >3 lbs in one week
- Squat Set 1 RPE 8+ when target is 6–7

7. Close with one line: what's looking strong and one thing to watch.
